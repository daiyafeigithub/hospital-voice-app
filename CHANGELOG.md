# Hospeech 更新日志

## v2.0 - 全新侧边栏设计 (2026-06-08)

### ✨ 新增功能

#### 1. 应用重命名
- ✅ 应用名称从"医院智能助手"改为 **"Hospeech"**
- ✅ 更简洁、专业的品牌形象
- ✅ 渐变色 Logo 设计（蓝色到靛蓝色）

#### 2. 侧边栏历史记录
- ✅ **滑动侧边栏**：从左侧滑出，宽度 320px
- ✅ **图标触发**：点击右上角时钟图标展开/收起
- ✅ **遮罩层**：半透明黑色背景，点击关闭
- ✅ **平滑动画**：300ms 过渡效果
- ✅ **独立滚动**：历史记录区域可独立滚动

#### 3. 固定布局
- ✅ **禁止主页面滚动**：`h-screen` + `overflow-hidden`
- ✅ **居中内容**：语音按钮和可视化始终居中
- ✅ **响应式设计**：适配各种屏幕尺寸

### 🎨 界面优化

#### 视觉改进
- **渐变背景**：slate-50 → blue-50 → indigo-50 三色渐变
- **更大按钮**：录音按钮从 128px 增加到 160px
- **双层脉冲**：录音时两个同心圆脉冲动画
- **阴影增强**：彩色阴影效果（shadow-blue-300/50）
- **标题渐变**：Hospeech 文字使用渐变色

#### 交互优化
- **图标按钮**：历史记录按钮改为纯图标，悬停变色
- **状态激活**：打开历史时图标变为蓝色高亮
- **工具提示**：鼠标悬停显示"查看/关闭历史记录"
- **对话计数**：主界面显示已有对话数量提示

#### 侧边栏设计
```
┌─────────────────────────┐
│ 历史记录          [×]   │ ← 头部
├─────────────────────────┤
│                         │
│  👤 用户消息            │
│  ⏰ 时间戳              │
│                         │
│  🤖 AI 回复             │
│  ⏰ 时间戳              │
│                         │
│  (可滚动区域)           │
│                         │
├─────────────────────────┤

└─────────────────────────┘
```

### 📱 用户体验

#### 主要改进
1. **更专注的界面**：移除主界面的对话列表，专注于语音交互
2. **清晰的历史访问**：通过侧边栏统一管理所有历史
3. **优雅的动画**：所有过渡都流畅自然
4. **触控友好**：大按钮、清晰的视觉反馈

#### 操作流程
```
1. 打开页面 → 看到居中的录音按钮
2. 点击录音 → 波浪动画 + 红色脉冲
3. 说话 → 实时音频可视化
4. 停止 → 自动识别并回答
5. 查看历史 → 点击右上角图标
6. 浏览历史 → 侧边栏滑出
7. 关闭历史 → 点击图标或遮罩层
```

### 🔧 技术实现

#### 侧边栏组件
```typescript
<aside 
  className={`fixed top-0 left-0 h-full ... ${
    showHistory ? 'translate-x-0' : '-translate-x-full'
  }`}
  style={{ width: '320px' }}
>
  {/* 侧边栏内容 */}
</aside>

{/* 遮罩层 */}
{showHistory && (
  <div 
    className="fixed inset-0 bg-black/20 backdrop-blur-sm z-40"
    onClick={() => setShowHistory(false)}
  />
)}
```

#### 固定布局
```typescript
<div className="h-screen ... overflow-hidden">
  <header>...</header>
  <main className="flex-1 flex items-center justify-center overflow-hidden">
    {/* 居中内容 */}
  </main>
</div>
```

### 🎯 设计理念

1. **简约至上**：减少干扰，突出核心功能
2. **空间利用**：侧边栏不占用主界面空间
3. **视觉层次**：清晰的 z-index 层级管理
4. **流畅体验**：所有动画都经过精心调优

### 🚀 性能优化

- ✅ CSS transform 硬件加速
- ✅ backdrop-blur 现代浏览器支持
- ✅ 按需渲染侧边栏内容
- ✅ 内存泄漏防护（音频资源释放）

### 📝 注意事项

1. 侧边栏宽度固定为 320px
2. 移动端会自动适配
3. 历史记录最多显示所有对话
4. 清空操作不可恢复

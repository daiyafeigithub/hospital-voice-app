# 百度语音识别配置指南

## 1. 注册百度智能云账号

访问：https://console.bce.baidu.com/

如果没有账号，需要先注册并实名认证。

## 2. 创建语音技术应用

1. 进入控制台：https://console.bce.baidu.com/ai/#/ai/speech/app/list
2. 点击"创建应用"
3. 填写应用信息：
   - 应用名称：医院智能助手（或任意名称）
   - 应用类型：选择适合的类型
   - 接口选择：勾选"语音识别"
4. 点击"立即创建"

## 3. 获取 API 密钥

创建成功后，在应用列表中可以看到：
- **App ID** (应用ID)
- **API Key** 
- **Secret Key**

## 4. 配置环境变量

将获取的密钥填入 `.env.local` 文件：

```env
BAIDU_APP_ID=你的App_ID
BAIDU_API_KEY=你的API_Key
BAIDU_SECRET_KEY=你的Secret_Key
```

## 5. 免费额度

- **每日免费调用次数**：50,000 次
- **并发限制**：根据账户等级不同
- **音频时长限制**：单次最长 60 秒

对于个人项目和小型应用完全够用！

## 6. 重启开发服务器

配置完成后，重启开发服务器：

```bash
npm run dev
```

## 7. 测试语音功能

1. 在浏览器中打开 http://localhost:3000
2. 点击麦克风按钮
3. 允许浏览器访问麦克风
4. 说话测试
5. 再次点击停止录音

## 常见问题

### Q: iOS Safari 可以使用吗？
A: ✅ 可以！使用 MediaRecorder API 录制音频，所有现代浏览器都支持。

### Q: 录音时长限制？
A: 当前设置为 10 秒自动停止，可以在代码中调整。

### Q: 识别准确率如何？
A: 百度语音识别对中文普通话的准确率很高，适合医疗场景。

### Q: 超出免费额度怎么办？
A: 按量付费，价格很便宜（约 2.5元/小时音频）。也可以申请提高免费额度。
